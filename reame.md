# Emotion Detection System 🎭

## 📌 Description
This project is a real-time emotion detection system developed using Python and OpenCV. It captures video from a webcam and analyzes facial expressions to detect human emotions.

## 🚀 Features
- Real-time face detection
- Emotion recognition from facial expressions
- Uses webcam input
- Simple and user-friendly

## 🛠️ Technologies Used
- Python
- OpenCV
- NumPy

## ▶️ How to Run
1. Install required libraries:
   pip install opencv-python numpy

2. Run the main file:
   python main.py

## 📂 Project Structure
- `main.py` → Main program
- `models/` → Trained model files
- `assets/` → Images/screenshots
